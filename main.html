CSS BOX MODEL

What is a css box model

is the essentialy a box that wraps around every html element

what are the main properties of a css box model

  • Border is the line that goes around the padding and content
  • Padding it clears an area around the content
  • Margin it clears an area outside the border
  • Content is wher text and images appears
  • Width specifies the width of the element's content area

calculate the total dimension of a css box model given the following:
width:20px,padding:400px,margin:20px,border:2px solid #EFE

400px(width) +20px(right+left padding) +20px(right+left margin) +2px(right+left border) =444px
